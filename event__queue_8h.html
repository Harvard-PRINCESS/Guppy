enum event_queue_mode

What mode does an event queue operate in?

Enumerator
EVENT_QUEUE_CONTINUOUS 

Run events continuously, as the waitset allows.

EVENT_QUEUE_ONESHOT 

Trigger one event at a time, when event_queue_trigger() is called.
