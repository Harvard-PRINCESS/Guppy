USB Generic Descriptor

This descriptor is used to supply the device driver with all the necessary information upon initialization and connection to the usb manager.

A request for a configuration descriptor returns the configuration descriptor, all interface descriptors, and endpoint descriptors for all of the interfaces in a single request. The first interface descriptor follows the configuration descriptor. The endpoint descriptors for the first interface follow the first interface descriptor.
