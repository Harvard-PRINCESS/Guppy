Detailed Description

USB Hub Port Status (USB Specification, Rev 2.0, Table 11-19 )

The hub status struct is returned when querying a hub port with the usb_hub_get_port_status() request.

The bit locations in the wPortStatus and wPortChange fields correspond in a one-to-one fashion where applicable.

USB Specification Rev 2.0, Section 11.24.2.7.1, gives a detailed description
